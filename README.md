# Pharmacovigilance pipeline based on FAERS data

### Pharmacovigilance teams in the private companies or the government need to monitor adverse events reports to identify potential drug safety hazzards, analyze adverse events trends and effectively investigate separate cases.

## Problem
Lots of raw data makes it hard to see trends and investigate adverse events

## Business questions:
- Which drugs show increasing reporting trends over the last 5 years?
- Which adverse events are most common over the last 5 years?
- Which outcome categories are growing over the last 5 years?

### This project ingests FDA FAERS data, transforms multi-table adverse event reports into an analytics-ready warehouse, and provides dashboard with data on drug safety trends over time.
